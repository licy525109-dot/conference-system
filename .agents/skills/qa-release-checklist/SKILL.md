# QA Release Checklist

## Review focus

1. Amount tampering
2. Order status transitions
3. Payment callback idempotency
4. Registration duplication
5. Admin authentication
6. Secret leakage
7. Missing tests
8. MVP scope creep
9. Personal data exposure

## Output format

- High risk issues
- Medium risk issues
- Low risk improvements
- Files to fix
- Tests to run
- Manual acceptance steps
