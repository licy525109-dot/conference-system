## 强制要求

1. 这些素材不是让 Codex 把整张截图直接铺上去，而是让 Codex 作为固定模板中的业务图片/图标资源使用。
2. 模板布局、文字层、按钮、卡片、列表、状态、交互都必须是程序化实现，不允许截图切图冒充页面。
3. 这些模板必须是**固定业务模板**，运营只能改字段配置，不允许自由拖拽到完全变形。
4. 后台手机预览、H5、小程序真机要共用同一套模板渲染协议，达到尽量一致。
